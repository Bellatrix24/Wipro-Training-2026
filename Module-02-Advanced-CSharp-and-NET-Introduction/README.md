# Module 02: Advanced C# and Introduction to .NET

## Overview
This module represents the bridge between core C# logic and professional web application development. It focuses on mastering the **ASP.NET Core** ecosystem, emphasizing architectural excellence, security, and scalable design patterns.

## Objectives
*   **Framework Mastery**: Deep dive into the ASP.NET Core runtime and internal architecture.
*   **Secure-by-Design**: Implement industry-standard security protocols including Hashing, Middleware-based authentication, and Input sanitization.
*   **Production Readiness**: Shift focus from functional "lab" scripts to robust, deployable web services that prioritize reliability and observability.
*   **Asynchronous Architecture**: Leveraging the TPL (Task Parallel Library) to build highly responsive and non-blocking applications.

## Technical Stack
*   **Framework**: .NET 8.0 / ASP.NET Core
*   **Language**: C# (Advanced Features)
*   **Security**: SHA-256/Argon2 Hashing, JWT/Cookie Auth, HSTS
*   **Architecture**: Middleware Pipelines, Dependency Injection (DI)
