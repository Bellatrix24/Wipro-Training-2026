# .NET CLI Cheat Sheet

Here are the basic commands I used today to create and run my projects:

| Command | Description |
| :--- | :--- |
| `dotnet new console` | Creates a new Console Application template. |
| `dotnet new mvc` | Creates a new Web App using the Model-View-Controller pattern. |
| `dotnet new webapi` | Creates a new RESTful Web API project. |
| `dotnet run` | Compiles and runs the project in the current directory. |

*Tip: Use `dotnet --help` if you forget any command!*
