# Day 3: Methods and Reference Types

## Summary
Today was a very productive day in our .NET training at Wipro! We focused deeply on **C# Methods** and **Reference Types**. 

Key things I learned:
- How to define methods to make code reusable.
- The difference between Value Types and Reference Types (and how memory is managed).
- Passing parameters to methods.
