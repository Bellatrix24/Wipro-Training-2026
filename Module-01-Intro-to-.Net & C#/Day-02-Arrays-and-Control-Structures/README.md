# Day 2: Arrays and Control Structures

Today was very productive! I learned how to handle more than one piece of data at a time using **Arrays**. Instead of creating 5 different variables for 5 marks, I can just use one array.

I also learned how to control what the program does using:
*   **Loops**: To repeat tasks (like printing a list).
*   **If-Else**: To make decisions (like checking if a student passed).
*   **Switch Cases**: To handle different options efficiently (like ranking).

It's starting to feel like I'm really building logic now.
